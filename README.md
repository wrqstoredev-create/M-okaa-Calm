## 🚀 عن المشروع (About The Project)
هذا المشروع عبارة عن منصة تجارة إلكترونية (E-commerce) متطورة ومخصصة لمجتمع اللاعبين. يتيح المتجر للمستخدمين تصفح وشراء المنتجات الرقمية بأسعار تنافسية، مع توفير واجهة مستخدم عصرية، استجابة سريعة، ولوحة تحكم قوية لإدارة المنتجات والطلبات والعملاء بسلاسة.

## 💻 التقنيات المستخدمة (Tech Stack)
تم بناء هذا المشروع باستخدام أحدث التقنيات لضمان أفضل أداء وحماية:
* **Frontend:** React.js, TypeScript, Tailwind CSS
* **Backend & Database:** Node.js, Supabase, Firebase
* **Deployment:** Vercel / GitHub
